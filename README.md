### 🔮 The SynapSoul Ecosystem & Featured Builds

* 🛡️ **SynapSoul Sentinel:** An Autonomous AI Cloud SecOps Platform. Acts as a Zero-Trust AI SOC analyst that actively scans Terraform architectures, writes remediation patches, validates them in a local sandbox, and neutralizes live API data exfiltration and LLM prompt leaks in real time.
* 🧠 **SynapSoul Nexus:** An advanced GenAI engine designed for Open-Source Intelligence (OSINT) gathering and highly secure professional research.
* 🏃‍♂️ **SynapSoul Motion:** An AI-powered web and mobile application for real-time, full-body pose tracking utilizing TensorFlow.js and Capacitor.js.
* 🌍 **AMR-Sentinel:** An IoT-enabled One Health Surveillance System featuring a React.js dashboard and a Flutter-based computer vision scanner that digitizes water quality test strips, reducing testing costs by 90%+.
* 🕶️ **AR Virtual Tour Guide:** An interactive Augmented Reality application providing a seamless "scan-to-play" storytelling experience by overlaying dynamic multimedia onto physical landmarks using Unity and Vuforia.
